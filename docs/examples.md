# Examples

## Example 1

Input array: `[2, 7, 11, 15]`
Target: `9`
Output indices: `[0, 1]`

## Example 2

Input array: `[3, 2, 4]`
Target: `6`
Output indices: `[1, 2]`

## Example 3

Input array: `[3, 3]`
Target: `6`
Output indices: `[0, 1]`

## No Pair Case

If no valid pair exists, the current program prints:

`No valid pair found.`
