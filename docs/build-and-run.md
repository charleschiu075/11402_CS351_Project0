# Build and Run

## Requirements

- A C++17-compatible compiler (`g++` or `clang++`)

## Build Command

```bash
g++ -std=c++17 -Wall -Wextra -pedantic src/main.cpp -o two_sum
```

## Run Command

```bash
./two_sum
```

## Sample Session

Input:

```text
Enter number of elements: 4
Enter 4 integers: 2 7 11 15
Enter the target number: 9
```

Output:

```text
[0, 1]
```
